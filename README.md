#Javascript Arrays: Array-Methods

## Description
Practice with array methods: 
* join
* slice
* forEach 
* map 
* filter
* reduce

## How to run
In your terminal run:
```
http-server -p 9999
```
then navigate to [here](http://localhost:9999) in your browser.
